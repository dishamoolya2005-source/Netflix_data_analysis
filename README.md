 Netflix Data Analysis Project

 Overview

This project is based on analyzing movie data using Python. The goal is to explore the dataset and extract meaningful insights such as top-rated movies, popular genres, and revenue trends.

Technologies Used

* Python
* Pandas
* NumPy

 Dataset

The dataset contains information about movies including:

* Movie Name
* Genre
* Rating
* Votes
* Revenue
* Director

Steps Performed

1. Imported required libraries
2. Loaded the dataset
3. Cleaned the data (handled missing values)
4. Explored the dataset using functions like `head()`, `info()`, `describe()`
5. Performed data analysis:

   * Filtered movies based on ratings
   * Sorted data by revenue and votes
   * Identified top-performing movies

Key Insights

* Identified highest-rated movies
* Found most popular genres
* Analyzed revenue patterns

Conclusion

This project helped in understanding how to clean, analyze, and extract insights from real-world datasets using Python.
